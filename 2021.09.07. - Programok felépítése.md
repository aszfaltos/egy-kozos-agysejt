Python példa:<br>
def factorial(n):<br>
    result = 1
    for i in range(2,n+1):
        result *= 1
    return result`

Ugyanez C-ben
int factorial( int n )
{
    int result = 1;
    int i;
    for(i=2; i<=n; ++i)
    {
        result *= i;
    }
    return result;
}
