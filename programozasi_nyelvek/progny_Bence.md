##  C-ből a Java-ba

- C alapú szintaxis
- Objektumelvű
- Erősen típusos
- Platform független
- Erőforrásintenzív
- Gazdag infrastruktúra
- alacsonyszíntű nyelv: bájtkód
- sok nyelv fordítható rá
- dinamikus szerkesztés
- kódmobilitás
- Java Virtual Machine Specification

**példakód**

```Java
class HelloWorld {
  public static void main( String[] args ){
    System.out.println("Hello world!");
  }
}
```
